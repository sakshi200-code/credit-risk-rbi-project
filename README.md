
#  Credit Risk Analytics

An end-to-end Credit Risk Analytics project that predicts borrower default risk using machine learning. The project demonstrates a complete credit risk workflow, from data preprocessing and exploratory data analysis to predictive modeling, risk segmentation, model evaluation, and stress testing.

---

##  Project Overview

Financial institutions rely on credit risk models to estimate the probability that a borrower will default on a loan. This project builds an interpretable machine learning pipeline to classify loan applicants into different risk categories and generate insights that support lending decisions.

The project covers:

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Credit risk prediction
- Model evaluation
- Risk segmentation
- Portfolio stress testing
- Business insights and recommendations

---

##  Objectives

- Analyze borrower and loan characteristics
- Predict loan default probability
- Identify key risk drivers
- Build an interpretable credit risk model
- Evaluate model performance using industry metrics
- Simulate portfolio performance under stress scenarios

---

## Dataset

The dataset contains borrower demographics, loan information, employment history, income, credit history, and loan status.

Main features include:

- Borrower age
- Annual income
- Employment length
- Home ownership
- Loan amount
- Loan purpose
- Interest rate
- Loan grade
- Credit history length
- Previous default history

Target Variable:

**loan_status**

- 0 → Non-default
- 1 → Default
